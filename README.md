# 📊 Excel Dashboard Project

## 📌 Project Overview

This project demonstrates how to analyze and visualize data using
Microsoft Excel. It includes data analysis techniques such as Pivot
Tables, Charts, Conditional Formatting, and Linear Regression to create
an interactive dashboard.

The goal of this project is to transform raw data into meaningful
insights and visual reports.

------------------------------------------------------------------------

## 🛠 Tools Used

-   📗 Microsoft Excel
-   📊 Pivot Tables
-   📈 Charts & Graphs
-   🎨 Conditional Formatting
-   📉 Linear Regression Analysis
